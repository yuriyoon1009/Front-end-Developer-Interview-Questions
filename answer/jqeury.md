## jquery
- jQuery() === $() (shorthand)
- attr() HTML의 속성을 취급
- prop() Javascript 프로퍼티를 취급. (get the value of a property)
```
var $input = $("input");

$input.attr("checked"); // "checked"
$input.prop("checked"); // true
```
